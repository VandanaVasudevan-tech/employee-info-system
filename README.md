# Django Employee Directory

A simple Django application that allows a company to maintain and display its list of employees.  
Employee data is added using the Django Admin panel, and the application displays:

- A homepage  
- A team page listing all employees  
- A detailed page showing full information about each employee  

This is a beginner-friendly Django project demonstrating models, templates, URL routing, and media handling.
